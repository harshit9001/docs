## Get All Recordings

Retrieve all recordings for the authenticated user.

<ParamField body="none" path="none" query="none">

</ParamField>

```bash
curl -X GET https://api.waterrai.com/recordings \
  -H "Authorization: Bearer YOUR_API_SECRET_OR_JWT_TOKEN"
```

```200:json
[
  {
    "id": 201,
    "meeting_id": 789,
    "url": "https://storage.example.com/recordings/abc123.mp4",
    "duration": 1800,
    "status": "completed",
    "created_at": "2024-01-15T11:10:00Z"
  }
]
```

## Get Recording by ID

Retrieve a specific recording by its ID.

<ParamField body="none" path="id" type="integer" required>
  The recording ID
</ParamField>

```bash
curl -X GET https://api.waterrai.com/recordings/201 \
  -H "Authorization: Bearer YOUR_API_SECRET_OR_JWT_TOKEN"
```

```200:json
{
  "id": 201,
  "meeting_id": 789,
  "url": "https://storage.example.com/recordings/abc123.mp4",
  "duration": 1800,
  "status": "completed",
  "created_at": "2024-01-15T11:10:00Z",
  "Transcript": {
    "id": 301,
    "meeting_id": 789
  }
}
```

## Get Recordings by Meeting ID

Retrieve all recordings for a specific meeting.

<ParamField body="none" path="meetingId" type="integer" required>
  The meeting ID
</ParamField>

```bash
curl -X GET https://api.waterrai.com/recordings/meeting/789 \
  -H "Authorization: Bearer YOUR_API_SECRET_OR_JWT_TOKEN"
```

```200:json
[
  {
    "id": 201,
    "meeting_id": 789,
    "url": "https://storage.example.com/recordings/abc123.mp4",
    "duration": 1800,
    "status": "completed"
  }
]
```
